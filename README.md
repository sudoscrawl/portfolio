# Designer-Developer Portfolio

_An ultra-sleek, modern, and architectural personal portfolio._

## 🎨 Aesthetic & Design

- **Ultra-Sleek & Modern:** Minimalist, architectural layout with focus on negative space.
- **Absolute Black Theme:** Depth created through a dark aesthetic with 1px border systems.
- **Monochrome & Purple:** Deep black/white theme with refined purple accents (`#8B5CF6`) used sparingly as category markers and guides.
- **Human-Centric:** Simple, direct, and non-commercial tone.

## 🏗️ Structure

- **Navbar:** Thinned, high-end sticky navigation with a functional hamburger menu for mobile.
- **Hero:** Left-aligned, balanced typography with a focus on understated elegance.
- **Bio:** Direct, human-centric story of a young developer-designer.
- **Work:** Minimal grid of selected projects with GITHUB links.
- **Connect:** Sleek contact form for discussions and research collaborations.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **SCSS:** Architectural styling with variables and nested rules.
- **JavaScript:** Custom mobile menu logic and intersection observers for subtle reveal animations.

## 🚀 Getting Started

### 1. Build Styles

To compile the SCSS files into CSS:

```bash
npm install
npm run build:sass
```

Alternatively, using `sass` globally:

```bash
sass scss/main.scss css/main.css
```

### 2. View Site

Simply open `index.html` in your browser.
