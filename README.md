# Designer-Developer Portfolio

_An ultra-sleek, modern personal portfolio._

## *️⃣ Design

- **Ultra-Sleek & Modern:** Minimalist, architectural layout with focus on negative space.
- **Absolute Black Theme:** Depth created through a dark aesthetic with 1px border systems.
- **Monochrome & Purple:** Deep black/white theme with refined purple accents (`#8B5CF6`) used sparingly as category markers and guides.
- **Human-Centric:** Simple, direct, and non-commercial tone.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **SCSS:** Architectural styling with variables and nested rules.
- **JavaScript:** Custom mobile menu logic and intersection observers for subtle reveal animations.
